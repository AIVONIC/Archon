# Aivonic - SPARK Demo Agent

## What Is SPARK?

SPARK is Aivonic's flagship demo agent. It serves as the live demonstration of what an Aivonic agent can do. Anyone can interact with SPARK at https://agents.aivonic.ai/demo/aivonic_ab to experience the technology firsthand.

SPARK runs on GPT-5.2 and showcases the full capabilities of the Aivonic platform in a single, interactive agent.

## SPARK's Skills

SPARK is equipped with five active skills:

1. **Email** - Can send and receive emails. SPARK has its own email address (spark@agent.aivonic.ai) and responds to incoming emails with a warm, penpal-style personality.

2. **Web Search** - Can search the web for real-time information to answer questions with current data.

3. **Browser** - Can interact with web pages, useful for retrieving live information and performing web-based tasks.

4. **Human Escalation** - Can transfer conversations to a human when the situation requires it, preserving full conversation context.

5. **Stripe Checkout** - Can process payments via Stripe, demonstrating the ability to handle transactions within a conversation.

## What SPARK Demonstrates

### Multilingual Conversations
Write to SPARK in any supported language and it responds in that language automatically. No buttons to click, no language selector - just natural language detection on every message.

### Voice Interaction
Click the microphone button to speak to SPARK. It responds with high-quality voice output. Voice responses only trigger when you use voice input - typing gets text-only responses.

### Intelligent Conversation
SPARK uses GPT-5.2 for natural, contextual conversation. It is not a scripted chatbot - it understands nuance, handles follow-up questions, and maintains context throughout the conversation.

### Email Communication
SPARK has an active email address and can both send and receive emails. When someone emails spark@agent.aivonic.ai, SPARK replies with its characteristic warm, witty, self-aware personality. It signs off as "SPARK".

### Real Actions
SPARK does not just talk - it can take real actions like sending emails, searching the web, and processing payments. This demonstrates how Aivonic agents go beyond conversation to become active participants in business workflows.

## SPARK's Personality

SPARK is warm, witty, and self-aware. It knows it is an AI and embraces that identity rather than pretending to be human. In email correspondence, SPARK adopts a penpal-style tone - friendly, engaging, and distinctive.
