# Aivonic - Frequently Asked Questions

## General

**What is Aivonic?**
Aivonic is an AI agent development company. We build, deploy, and manage custom AI agents for businesses. We are not a DIY chatbot platform - we build the agent for you, tailored to your business.

**How is this different from a chatbot?**
Traditional chatbots follow scripted decision trees and forget conversations after the session ends. Aivonic agents use large language models (LLMs) for natural conversation, remember every interaction permanently, can take real actions (send emails, book appointments, process payments), and pass a quality certification before going live.

**Do I need technical skills to use Aivonic?**
No. We handle all the technical work. Your total time investment is approximately one hour for a discovery call and providing onboarding information. We build, test, certify, and deploy the agent for you.

**How long does it take to get an agent live?**
For straightforward deployments, an agent can be production-ready the same day. More complex setups with multiple integrations or large knowledge bases may take a few days.

## Pricing and Plans

**How much does it cost?**
Setup fees range from $5,000 (Standard Agent) to $18,000 (Enterprise Multi-Agent). Monthly fees are $1,000/mo for Standard support or $1,500/mo for Premium support. See the full pricing breakdown in our products and pricing documentation.

**What is the difference between Standard and Premium support?**
Standard support includes email support with a 24-hour response time. Premium support includes priority support with a 4-hour response time and a dedicated Slack channel.

**Is there a free trial?**
We offer a free 30-minute discovery call where we discuss your needs and determine if Aivonic is a good fit. You can also interact with our demo agent SPARK at https://agents.aivonic.ai/demo/aivonic_ab to see the technology in action.

**What is the difference between single-agent and multi-agent plans?**
Single-agent plans deploy one AI agent for your business. Multi-agent plans deploy multiple coordinated agents that work together, useful for businesses with complex workflows spanning different departments or functions.

## Technical

**What languages does the agent support?**
The agent supports multilingual conversations with automatic language detection. Customers can write or speak in their language, and the agent responds in the same language. No configuration is needed.

**Can the agent handle voice conversations?**
Yes. Customers can speak to the agent using their browser's built-in microphone. The agent responds with high-quality voice output via ElevenLabs or Hume AI. Voice responses only activate when the customer uses voice input - text conversations stay text-only.

**Does the agent remember past conversations?**
Yes. Aivonic agents have infinite memory with 50M+ message capacity. Every interaction is remembered permanently, so customers never need to repeat themselves.

**What can the agent actually do besides chat?**
Agents have access to 31 built-in skills including email delivery, appointment booking, CRM integration, order tracking, payment processing (Stripe), web search, document generation, human escalation, and more. The specific skills assigned depend on your business needs.

**How does the agent get knowledge about my business?**
We ingest your business information through our Archon knowledge base system. This can include website content (crawled automatically), uploaded documents, FAQs, product catalogs, and other materials. The agent uses this knowledge to answer customer questions accurately.

**What AI models do you use?**
We support OpenAI models (including GPT-5.2) and Anthropic Claude models. The model is selected based on the agent's requirements and use case.

**Is the agent secure?**
Yes. Every agent is protected by the Crust security gateway, which provides Data Loss Prevention (DLP) and prompt injection protection. Security testing is part of the mandatory 4-phase evaluation every agent undergoes before deployment.

## Deployment

**How do I install the chat widget on my website?**
We provide a single script tag to add to your website. The widget is fully customizable - you can set colors, logo, title, and choose between dark and light themes.

**Can I customize the agent's appearance?**
Yes. The chat widget supports custom colors, logo, title text, and dark/light themes. Branding is configured during deployment and can be adjusted through the client dashboard.

**What if the agent cannot answer a question?**
Agents include a human escalation skill that transfers the conversation to a human agent with full context. You define the escalation conditions during setup.

**Do I get analytics on agent performance?**
Yes. The client dashboard includes analytics, conversation history, and real-time support monitoring. You can see how the agent is performing and review conversations at any time.

## Voice AI Agent (Separate Product)

**What is the Voice AI Agent?**
The Voice AI Agent is a separate, dedicated product built on Vapi AI with a Make.com backend. It is designed for voice-first use cases like phone systems and voice assistants. This is different from the voice support feature that comes with every portal agent.
